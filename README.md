### To run your own server please follow the guide:

## Linux/Windows:
- Clone the repo
- run ```pip install -r requirements.txt```
- start the api with ```python api.py```

## MacOS:
- Clone the repo
- run ```pip3 install -r requirements.txt```
- start the api with ```python3 api.py```
